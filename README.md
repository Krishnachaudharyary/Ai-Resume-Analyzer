# AI Resume Analyzer

A standalone browser app for analyzing resumes, matching them against job descriptions, checking ATS readiness, finding skill gaps, suggesting improvements, and ranking multiple candidates.

## Open the project

Open `index.html` in a browser.

## Features

- Upload PDF, DOCX, or TXT resumes
- Extract resume text in the browser
- Parse name, email, phone, skills, education, experience, certifications, projects, and links
- Score resumes from 0 to 100
- Check missing sections, grammar signals, ATS compatibility, experience level, keyword density, and action verbs
- Compare a resume with a job description
- Show matching skills, missing skills, and learning recommendations
- Rewrite weak resume statements into stronger bullet points
- Upload multiple resumes and rank candidates

## Notes

PDF and DOCX extraction uses browser libraries from public CDNs. If a browser blocks those libraries, paste the resume text into the text box and run the analyzer.
